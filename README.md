# 🏔️ Tèxtils Montblanc - Web E-commerce + Factory 4.0

Prototip funcional d'una web moderna per a una empresa de roba de muntanya amb integració d'e-commerce, sistema de pagament i control d'estoc en temps real.

## ✨ Característiques Principals

### 🎨 Diseño Moderno
- **Estilo**: Minimalista de luxe (inspirado en Apple, The North Face, Patagonia)
- **Colores**: Gris oscuro (#1f2937), Blanco puro, Azul eléctrico (#1e90ff)
- **Layout**: Fluid, responsive 100% (móvil, tablet, PC)
- **Efectos**: Scroll suave, tarjetas con sombras modernas, animaciones fluidas

### 📱 Secciones Incluidas

#### 1. **Header Sticky**
- Menú de navegación pegajoso con blur (backdrop filter)
- Logo "TM" con gradiente
- Enlaces a: Inicio, Tienda, Fábrica 4.0, Contacto
- Menú mobile responsive

#### 2. **Hero Section**
- Imagen de fondo con gradiente oscuro (efecto montaña)
- Texto grande: "El futur de la muntanya és digital"
- Botón "Ver Catálogo" con hover effects
- Animaciones de entrada suave (fade-in)

#### 3. **Tienda (E-commerce)**
- Grid de 3 productos (responsive: 1 col móvil, 2 col tablet, 3 col desktop)
- Productos:
  - **Jaqueta Gore-Tex** (€189) - Stock Disponible ✓
  - **Pantalons Tècnics** (€129) - Limited Edition
  - **Botas de Montaña** (€249) - Stock Disponible ✓
- Cada tarjeta con:
  - Imagen representativa
  - Badge dinámico de disponibilidad
  - Descripción
  - Precio y valoración ⭐
  - Botón "Añadir a la cistella"
- Hover effects modernos

#### 4. **Checkout / Proceso de Pago**
- **Resumen de compra** (sticky panel):
  - Desglose de items
  - Subtotal automático
  - Costo envío (€9.99)
  - Total actualizado
- **Formulario de pago**:
  - Opción: Tarjeta de Crédito
  - Opción: Bizum (con teléfono)
  - Animación de procesamiento
  - Spinner de carga
  - Confirmación de compra

#### 5. **Fábrica 4.0 Dashboard**
- **Métricas en tiempo real** (actualización cada 5 segundos):
  - 🌡️ Temperatura de Horno
  - ⚙️ Máquinas en funcionamiento
  - 📦 Prendas producidas hoy
  - ✅ Control de calidad
- **Gráfico interactivo** (Chart.js):
  - Producción últimas 24 horas
  - Valores dinámicos
- **Activity Log** + Estadísticas del turno

#### 6. **Footer**
- Info de empresa
- Enlaces rápidos
- Servicio al cliente
- Contacto (Igualada)
- Redes sociales
